Homotopy Type Theory✓ reviewed

Logic · Foundations  ·  ~4 min read

Core Concept

INFO
Homotopy Type Theory (HoTT) is a foundational framework that interprets the types of Martin-Löf dependent type theory as spaces (in the homotopy-theoretic sense), terms as points in those spaces, and crucially, proofs of equality as paths between points.

The pivotal reinterpretation is of the identity type. In ordinary type theory, Id_A(a, b) is a proposition asserting a = b. In HoTT, it becomes the path space between a and b in the space A — paths can themselves have higher paths between them (homotopies between homotopies), giving rise to an intrinsically ∞-groupoid structure.

The Univalence Axiom, due to Voevodsky, is the engine of the whole enterprise:

$$ (A \simeq B) \simeq (A =_{\mathcal{U}} B) $$

An equivalence of types is an identification in the universe $\mathcal{U}$. This collapses a longstanding gap between "isomorphic" and "equal" and makes type theory invariant under equivalence by construction.

Higher Inductive Types (HITs) extend ordinary inductive types by allowing constructors that produce inhabitants of identity types — not just points, but paths, 2-paths, etc. The circle $S^1$, for instance, is defined by a point base : S^1 and a path loop : base = base.

Why It Matters

HoTT sits at the intersection of three deep subjects:

  • Foundations of mathematics: It offers an alternative to set-theoretic foundations (ZFC) where equality is structural, not extensional set membership. Univalence encodes the mathematical practice of treating isomorphic objects as interchangeable.
  • ∞-category theory: HoTT provides an internal language for (∞,1)-toposes, much as ordinary type theory is the internal language of locally cartesian closed categories. This connection, via Grothendieck's homotopy hypothesis, links syntactic proof theory to deep geometry.
  • Verified mathematics: The entire framework is machine-checkable. Proofs of classical results (e.g., van Kampen's theorem, the Blakers–Massey theorem) have been formalized in Coq and Agda, often revealing new synthetic proofs shorter than classical ones.
TIP
Univalence is not provable in MLTT — it requires a new axiom. But it is consistent, as witnessed by the simplicial set model of Voevodsky and the cubical set models. This means it adds genuine proof-theoretic strength.

Key Details

  1. ∞-Groupoid Semantics: Every type A carries the structure of an ∞-groupoid: points are 0-cells, paths are 1-cells, paths between paths are 2-cells, etc. Types with trivial higher structure are sets (h-sets); types with at most one inhabitant are propositions (h-props). This stratification is the homotopy level (h-level).
  1. Truncation: The $n$-truncation $\|A\|_n$ kills homotopy above level $n$. The (-1)-truncation $\|A\|_{-1}$ converts any type into the mere proposition of its inhabitedness — recovering classical logic as a shadow.
  1. Cubical Type Theory: A computationally well-behaved variant (CCHM/Cubical Agda) gives univalence a definitional computational interpretation using an interval primitive $\mathbb{I}$, resolving the original HoTT book's use of univalence as an axiom with no reduction rule.
  1. Synthetic Homotopy Theory: Classical results like $\pi_1(S^1) \cong \mathbb{Z}$ and the Hopf fibration are proved synthetically — inside the type theory, without building topological spaces explicitly. The proofs are often more elegant and illuminating.
  1. Propositional vs. Definitional Equality: HoTT sharpens the distinction. Definitional equality is invisible to the type-checker; propositional equality is a type. Cubical variants blur this by making more equalities definitional via transp and hcomp primitives.

Current State

WARNING
Note: this information may be dated

Active frontiers include: completing the Homotopy Type System (HTS) that mixes strict and weak equality; formalizing the chromatic tower and stable homotopy theory internally; developing modalities for cohesion and differential cohesion (Schreiber's program in Modal HoTT); and scaling proof assistants (Agda, Lean 4 with Mathlib) to handle large synthetic topology libraries. The question of a fully satisfactory ∞-topos-internal language remains partially open.

Discussion Prompts

  1. Univalence makes "isomorphic implies equal" a theorem schema — but does this dissolve or merely relocate the philosopher's question of identity? What does it mean for two types to be "the same"?
  2. Cubical type theory restores computational content to univalence. Does this suggest that the original HoTT axiom was always a meta-theoretical stopgap, or is the cubical interval itself philosophically loaded?
  3. Synthetic homotopy theory proves results about $\pi_n(S^m)$ without building spaces — the logic is the geometry. What are the limits of this synthesis? Are there topological phenomena fundamentally resistant to type-theoretic internalization?
